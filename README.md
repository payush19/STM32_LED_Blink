# STM32 LED Blink

## Overview
This project demonstrates LED blinking using the STM32F446RE Nucleo development board. The firmware is developed in Embedded C using STM32CubeIDE.

## Features
- GPIO Configuration
- LED ON/OFF Control
- Delay Implementation
- Embedded C Programming

## Hardware Used
- STM32F446RE Nucleo Board

## Software Used
- STM32CubeIDE
- Embedded C

## Project Objective
To understand STM32 GPIO peripheral configuration and basic firmware development concepts by implementing an LED blinking application.

## Result
Successfully implemented LED blinking on STM32F446RE using GPIO control.
